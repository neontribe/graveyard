# Composer template for Drupal projects

## Install drupla 8 with nt8 ready to go:
```
composer \
    create-project neontribe/nt8-installer EntyAte \
    --stability dev \
    --no-interaction
```

## Chnage to the folder where the vagrant file is:

    cd EntyAte

## The run Vagrant up:

    vagrant up
